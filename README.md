# Supply-Chain-Analysis
Exploratory Data Analysis on Supply Chain dataset relevant  to GCC Manufacturing and Logistics companies.

## Insights Covered
- Product type wise revenue analysis
- Supplier wise defect rate analysis
- Shipping carrier performance — cost vs time
- Location wise manufacturing cost
- Stock levels vs Lead times relationship
- Transportation mode cost comparison
- Defect rate vs Inspection results

## ML Model
- Random Forest Classifier to predict defect rate
- Correlation analysis performed
- Weak feature relationships identified due to synthetic data

## Tools Used
Python, Pandas, Matplotlib, Sklearn

## Dataset
Supply Chain Dataset from Kaggle

## Target Companies
Caterpillar, Cummins, Honeywell, ABB GCC
